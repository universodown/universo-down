# universo-down
Projeto Universo Down - SENAC/SC

## Pré-requisitos
- **NodeJS** (14.18.0)
- **npm**

# Inicilizando o ambiente
```bash
npm install
npm run build
npm start
```

# Scripts - configurados no arquivo package.json
- `build`: Compila o código TypeScript do `src/` para o diretório `build/`
- `start`: Inicializa o servidor
